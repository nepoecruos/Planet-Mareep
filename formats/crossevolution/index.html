What is Cross Evolution?

Since the beginning of the first generation, Pokémon had access to evolution which unlocked new stats, moves, typing and eventually abilities. Every Pokémon has access to a unique evolution line which allows them to receive their own previously mentioned evolution unlockments. However, what if another Pokémon could receive those certain unique unlockments?

Here's an example:

Type: Steel / Ghost

Hitmontop (Honedge) @ Life Orb
Ability: Technician
EVs: 248 HP / 252 Atk / 8 Spe
Adamant Nature
- Shadow Sneak
- Bullet Punch
- Mach Punch
- Swords Dance

HP: 45 → 60 (+15)
Attack: 80 → 140 (+60)
Defense: 100 → 160 (+60)
Special Attack: 35 → 35 (+0)
Special Defense: 37 → 112 (+75)
Speed: 28 → 63 (+35)

BST: 325 → 570 (+245)

Honedge now inherits Hitmontop's stat changes, which typically would be applied to Tyrogue. It forgets it's old abilities for Intimidate, Technician and Steadfast which is what Hitmontop gets when it evolves. And it is able to utilize two Hitmontop moves such as Mach Punch and Rapid Spin. (The set above only uses one Hitmontop move)

Honedge is able to cross-evolve into Hitmontop because it is a first stage and Hitmontop is a second change. It would be unable to cross-evolve into third stage Pokémon because it is a first stage. However, it's second stage Doublade is able to cross-evolve into third stages!

Here's an example:


Type: Steel / Flying

Dragonite (Doublade) @ Leftovers
Ability: Multiscale
EVs: 252 HP / 252 Def / 4 SpD
Impish Nature
- Iron Head
- Toxic
- Defog
- Roost

HP: 59 → 89 (+30)
Attack: 110 → 160 (+50)
Defense: 150 → 180 (+30)
Special Attack: 45 → 75 (+30)
Special Defense: 49 → 79 (+30)
Speed: 35 → 45 (+10)

BST: 448 → 628 (+180)

Dragonair's second type changes to Flying when it evolves. So whenever another second stage Pokémon cross evolves with Dragonite their second typing is changed to Flying automatically. They obtain Inner Focus and Multiscale and 2 Moves from Dragonite. In this case, Doublade inherited Roost and Defog from it.

Doublade cannot cross-evolve into Hitmontop because it is a second stage Pokemon just like Hitmontop. So the rule of the game is, you can cross-evolve with the evolution that's higher than you by one stage. Hence, third stages cannot cross-evolve because fourth stages do not exist. And you are not allowed to cross-evolve into a mega evolution.

Additional Information:

- Second stage Pokemon who do not evolve cannot cross-evolve, for example Doublade can cross-evolve but Gliscor cannot.
- Items like Light Ball or Thick Club will only work on regular Pikachu and Marowak, not cross-evolved ones.
- Eviolite () does not work on cross-evolved Pokemon as they're treated as different and fully evolved species.
- Signature Z-Moves like Extreme Evoboost will not work if your Eevee is cross-evolved.
